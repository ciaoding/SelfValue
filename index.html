安装到主屏幕

PWA 需要通过 HTTPS 或 localhost 测试；直接打开本地文件不能完整启用离线缓存。

  1. 用 Safari 打开这个网页
  2. 点击分享按钮
  3. 选择“添加到主屏幕”
  4. 之后从桌面图标打开
  1. 用 Chrome 打开网页
  2. 点击菜单
  3. 选择“安装应用”或“添加到主屏幕”

直接用浏览器打开部署后的网址即可。手机和电脑默认各自保存本地数据，不会自动同步；迁移数据请使用 JSON 导出/导入。
